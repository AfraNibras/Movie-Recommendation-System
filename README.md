# 🎬 Movie Recommendation System

A simple **Movie Recommendation System** built using Python. This application recommends movies based on the user's preferred genre using Python dictionaries and lists. It was developed as part of an **Artificial Intelligence Internship** to understand the basics of recommendation systems.

---

## 📌 Features

- 🎥 Recommends movies based on genre
- 📚 Uses Python dictionaries and lists
- 🎯 Supports multiple movie genres
- 🔄 Allows multiple recommendations in one session
- ❌ Handles invalid genre inputs gracefully
- 🖥️ Clean and user-friendly terminal interface
- 🧩 Modular Python code

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/AfraNibras/Movie-Recommendation-System.git
```

### 2. Navigate to the project folder

```bash
cd Movie-Recommendation-System
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the project

```bash
python recommendation.py
```
